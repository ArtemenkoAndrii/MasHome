We built the "Mas Money" app as a free app. This application is provided by us at no cost and is intended for use as is. If you choose to use this application, then you agree to the collection and use of information in relation to this policy.

2. Information Collection and Use

The Mas Money app collects and stores all user-entered data on the user's device only and does not transfer it to any other servers or devices, nor does it share it with any third parties.

For more efficient use of the application, the user may allow the application to capture SMS messages and Push Notifications. These will be collected and processed the same way as user-entered data (not shared and not distributed with anyone).

3. Application Permissions

In order to use the application as efficiently as possible, the user may grant the following permissions:

  • android.permission.RECEIVE_SMS – to allow the application to capture SMS messages.
  • android.permission.QUERY_ALL_PACKAGES – to allow Mas Money to read the name of a Push Notification sender.

To activate the ability to capture Push Notifications, it is also necessary to allow "Device & app notifications" permission in the "Special app access" section.
