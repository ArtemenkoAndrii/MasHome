    Information Collection and Use

    The Mas Money app collects and stores all user-entered data on the user's device only and does not transfer it to any other servers or devices as well as doesn't share it with any 3rd parties. For more efficient use of the application, the user may allow the application to capture SMS messages and Push Notifications which will be collected and processed the same way as user-entered data(not shared and not distributed with anyone).

    Application permissions

    In order to use the application as efficiently as possible, the user may grant the following permissions:

  • android.permission.RECEIVE_SMS to allow the application to capture the SMS messages.
  • android.permission.BIND_NOTIFICATION_LISTENER_SERVICE to allow Mas Money to capture the Push Notifications.

    • To activate the ability to capture Push Notifications it is also necessary to allow "Device & app notifications" permission in the "Special app access" section.
